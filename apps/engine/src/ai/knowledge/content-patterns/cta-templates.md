## üéØ CTA Placement Strategy

### Beginning vs End

| Placement | Effect | Best For |
|-----------|--------|----------|
| End | Natural, earned | Standard |
| Middle | Rides peak engagement | Stories |
| Beginning | Only if strong hook | Rare |

### The Value-First Rule

```
1. Deliver value first
2. Create emotional peak
3. Insert CTA at peak
4. End with hook/loop
```

## üí° Advanced CTA Techniques

### The Choice CTA
```
"Comment A if [option] or B if [other option]"
Creates engagement through simple decision.
```

### The Challenge CTA
```
"Try this and tag me in your results"
Creates user-generated content loop.
```

### The Tease CTA
```
"Part 2 is even crazier - make sure you're following"
Uses curiosity to drive follows.
```

### The FOMO CTA
```
"This only works for the next [X] days"
Creates urgency without being spammy.
```
