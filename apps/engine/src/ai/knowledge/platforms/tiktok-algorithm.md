## ü™ù Hook Engineering

### The 1-Second Rule

TikTok users decide to watch or scroll within **1 second**. Hook strategies:

#### Pattern Interrupts
```
"POV: You just discovered..."
"Story time: [dramatic pause]"
"Wait, did you know that..."
"STOP. Before you scroll..."
```

#### Curiosity Gaps
```
"The thing nobody tells you about [X]..."
"I tested [thing] for 30 days. Results?"
"3 things you're doing wrong with..."
"This changed everything I knew about..."
```

#### Controversy Hooks
```
"Unpopular opinion: [statement]"
"I'm about to make people angry..."
"Everyone is lying about..."
```

### Pattern Interrupt Cadence

Every **2-3 seconds**, the viewer's attention needs a refresh:
- Visual change (zoom, cut, transition)
- Audio change (music drop, voice shift)
- Text overlay update
- New information reveal

## üîÑ Loop Engineering

### Why Loops Matter

When viewers rewatch, TikTok registers:
- 100%+ completion rate
- Extended watch time
- Higher engagement potential

### Loop Techniques

#### Seamless Loop
End frame connects visually/audibly to start frame.

#### Revelation Loop
"Wait, go back and watch the beginning again..."

#### Missing Detail Loop
Subtle detail in the beginning that's revealed at the end.

#### Question Loop
End with a question that's answered in the beginning.

## üìä Content Lifecycle

### The First 30-60 Minutes

Critical window for algorithm evaluation:
- Initial push to ~200-500 viewers
- Engagement rate determines next batch
- High performance ‚Üí exponential expansion

### Viral Trajectory

```
Phase 1 (0-1h):    Test batch (~500 views)
Phase 2 (1-4h):    Expansion if metrics good (~5K views)
Phase 3 (4-24h):   Viral push (~50K-500K views)
Phase 4 (24h+):    Sustained or decline
```

### Posting Time (Secondary Factor)

Best times vary by niche, but general peaks:
- 6-9 AM (morning scroll)
- 12-2 PM (lunch break)
- 7-11 PM (evening wind-down)

## üö´ Shadowban Triggers

### Actions That Limit Reach

- Violating community guidelines
- Excessive posting (10+/day)
- Deleting and reposting same content
- Using banned hashtags
- External watermarks (Instagram, YouTube)
- Political/controversial content in some regions

### Recovery Strategies

1. Pause posting for 24-48 hours
2. Post original, high-quality content
3. Engage authentically on other videos
4. Avoid all trigger behaviors
